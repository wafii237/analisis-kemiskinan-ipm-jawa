# Analisis Kemiskinan dan IPM Provinsi di Pulau Jawa (2015–2024)

## Latar Belakang
Kemiskinan dan Indeks Pembangunan Manusia (IPM) merupakan indikator penting
dalam mengukur kesejahteraan masyarakat. Pulau Jawa memiliki peran strategis
dalam perekonomian nasional sehingga analisis dinamika kemiskinan dan IPM
di wilayah ini menjadi relevan.

## Tujuan
- Menganalisis tren kemiskinan dan IPM provinsi di Pulau Jawa
- Membandingkan kondisi antar provinsi
- Mengkaji hubungan antara IPM dan tingkat kemiskinan

## Data
- Sumber: Badan Pusat Statistik (BPS)
- Periode: 2015–2024
- Unit analisis: Provinsi di Pulau Jawa

## Metodologi
- Data cleaning dan standarisasi data multi-tahun
- Penggabungan data kemiskinan dan IPM menjadi dataset panel
- Analisis deskriptif dan visualisasi data

## Insight Utama
- IPM meningkat secara konsisten di seluruh provinsi
- Tingkat kemiskinan cenderung menurun dengan lonjakan pada tahun 2020
- Hubungan IPM dan kemiskinan bersifat negatif namun tidak homogen antar provinsi

## Tools
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Catatan
Project ini bersifat analisis deskriptif dan tidak bertujuan untuk menguji
hubungan kausal antar variabel.
